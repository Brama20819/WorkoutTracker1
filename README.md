A full-stack fitness tracking application.It allows users to put it the name of the workout, the description and name of exercises and the weight of exercises to track the increase or decrease over time.


Tech Stack:Java(Spring boot),Angular,MySQL

Deployment:Railway Cloud

Features
Add, update, delete workouts

Track exercise weight progress over time

Responsive UI

Charts that visualize weight progression.


How to test this: 
1.Click the View as guest button (registration coming soon)

2.Add a workout:
   Enter Workout Name, Workout Description, Exercise Name, and Weight

3.Track exercise progress:
   Click View Exercise Name (e.g., Bench, Squat) to see the weight chart at the top

4.View all workouts:
   Click Show All Workouts to see every workout entered.
